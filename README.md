# Data Classification Using AI

## Objective
Build a classification model using the Iris dataset.

## Tools Used
- Python
- Scikit-Learn
- VS Code

## Algorithm Used
K-Nearest Neighbors (KNN)

## Steps Performed
1. Loaded Iris Dataset
2. Scaled Data
3. Split Data into Training and Testing Sets
4. Trained KNN Model
5. Made Predictions
6. Evaluated using Accuracy, Confusion Matrix and F1 Score

## Output
The model successfully classified flower species with high accuracy.
